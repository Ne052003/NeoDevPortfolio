Library Online

Bookstore API is a robust and scalable backend system designed to manage an online bookstore efficiently. The API provides essential functionalities, including user authentication, book management and billing. It follows best practices in software development, ensuring security, maintainability, and scalability through a modular architecture and industry-standard tools.

The project implements JWT-based authentication for secure user access, role-based authorization to distinguish between customers and administrators, and RESTful API design for seamless integration with frontend applications. Additionally, database interactions are optimized using Hibernate (JPA), and the system supports CRUD operations for books, users, and bills.

This API is built to be easily extendable, allowing future enhancements such as book cover image management, notifications, and advanced reporting.
